# Assistente de Meta

Este projeto é um assistente web para ajudar jogadores a obter dicas, estratégias e builds para jogos populares como Valorant, League of Legends e Rainbow Six Siege, utilizando a API Gemini AI do Google.

## Funcionalidades

- Consulta de estratégias e dicas para jogos selecionados
- Interface simples e responsiva
- Integração com a API Gemini AI
- Respostas formatadas em Markdown

## Como usar

1. Clone ou baixe este repositório.
2. Abra o arquivo `index.html` em seu navegador.
3. Insira sua API KEY do Gemini.
4. Selecione o jogo desejado.
5. Escreva sua pergunta e clique em **Perguntar**.

## Tecnologias

- HTML, CSS, JavaScript
- [Showdown.js](https://github.com/showdownjs/showdown) para conversão de Markdown
- API Gemini AI (Google)

## Observações

- É necessário possuir uma API KEY válida do Gemini AI para utilizar o assistente.
- As respostas são geradas automaticamente pela IA e podem variar em precisão.
